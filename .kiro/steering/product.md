# Net Worth Tracker - Product Overview

## What It Is
A secure, privacy-first financial portfolio management application that runs locally. Users can track investments across multiple account types with military-grade encryption and complete data privacy.

## Key Principles
- **Privacy First**: All data stays local, no cloud dependencies, zero telemetry
- **Security**: AES-256 encryption, PBKDF2 key derivation, localhost-only access
- **User-Friendly**: Browser-based interface, demo mode, cross-platform support

## Supported Account Types
- Certificate of Deposit (CD)
- Savings Accounts
- 401k Retirement
- Trading Accounts (with real-time stock prices)
- I-bonds

## Core Features
- Master password authentication
- Encrypted data storage
- Historical performance tracking
- Export/import functionality
- Demo mode with synthetic data
- Real-time stock price updates

## Target Users
Individual users managing personal financial portfolios who prioritize data privacy and local control over their financial information.
